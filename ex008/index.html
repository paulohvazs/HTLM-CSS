Principais formatações

Negrito/Destaque

nessa frase temos um termo em negrito usando a tag B (não semânica)

Já nessa, temos um termo em destaque usando a tag STRONG (semântica)

Itáligo/Enfâse

Nessa trafase temos um termo em itálico usando a tag I (não semântica)

Já nesse, temos um termo em ênfase usando a tag EM (semântica)

Texto marcado

Podemos deixar um texto marcado com a tag MARK. Para deixar todas as marcações em uma determinado cor, deve ser feito um estilo no campo head. Dessa forma a marcação terá a cor determinada

Deixar o texto grande ou pequeno

É possível deixar o texto grande ou pequeno .

Texto deletado

É possível marcar que um texto foi deletado.

Texto inserido

Da mesma forma é possível marcar que um texto foi inserido com a tag INS. Uma tag que deixar o texto com a mesma formatação seria a tag U (não semântica)
